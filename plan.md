### 회의 참여 UI

회의는 별도 회의실 포탈로 이동하는 구조가 아니다.

조건

- 같은 공간 안에 사용자 2명 이상이 가까이 모임
- 모인 사용자 중 최소 2명이 온라인 상태
- 회의 참여 가능 범위 안에 있음

활성화 UI

- "회의 참여하기" 버튼
- 현재 모인 사용자 수
- 참여 가능 사용자 목록

회의 참여 시

- 해당 위치 기반 임시 회의 세션 생성 또는 참가
- 음성채팅 활성화
- 참여자는 Presence가 회의중으로 변경

동시에 존재

- 사용자는 같은 공간에 계속 남아 있음
- 회의 UI는 조건이 해제되면 비활성화

회의 종료 시

- 회의 세션 자동 정리
- Presence 원복
- 회의 전용 공간 삭제 없음
